## Требования

- Go 1.26.1

## Makefile

```sh
make build
```

Собирает приложение в исполняемый файл `broker`.

```sh
make run
```

Собирает приложение и запускает `./broker $(PORT)`. Если `PORT` не указан,
используется `8080`.

```sh
make test
```

Собирает приложение, запускает локальный брокер и проверяет HTTP API через
`curl`: добавление сообщений, FIFO-чтение, пустые очереди, ошибку при
отсутствии параметра `v`, таймаут ожидания и порядок ожидающих клиентов.

```sh
make clean
```

Удаляет собранный бинарник `broker`.
